# Predictive Analysis of Financial Performance in the Automotive Industry

Project developed for the *Fundamentals of Business Management* course (MSc in Data Science and Business Informatics, University of Pisa).

The project focuses on the analysis and prediction of financial performance in the automotive sector, using Ferrari as the main case study.

It covers key topics such as financial forecasting, key performance indicators (KPIs), financial ratios, and competitive analysis, with the goal of supporting strategic decision-making and understanding market dynamics.
